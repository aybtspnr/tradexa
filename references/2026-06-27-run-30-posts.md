## Lessons Learned
1. **delegate_task write_file bug persistente:** Subagent `write_file` does NOT persist to the parent filesystem. Must use `cat heredoc` in terminal commands within subagent goals. This is a critical workflow requirement.
2. **Batch size 3×2 worked well** for production — each subagent with 2 posts completes in ~3-5 min.
3. **Safe-deploy.sh step 3 token issue:** auth.json token expired. Fallback to direct `npx vercel deploy --prod --yes --token "..."` worked.
4. **Shared file restoration between batches** was essential — multiple subagents modified postMeta.ts and postContentMap.ts despite explicit instructions not to.
5. **Integration script syntax:** Python f-strings with nested TypeScript braces require careful escaping. Use string concatenation for complex template entries.
